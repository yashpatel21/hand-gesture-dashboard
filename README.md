# Hand Gesture Dashboard

A smart dashboard controlled by hand gestures via webcam. Built with React, TypeScript, and MediaPipe.

**Features**

-   Displays date, time, weather, calendar, reminders, and music controls
-   Gesture recognition, including point-and-click gestures for cursor control of interactive elements, as well as swipe gestures for moving between screens
-   Visual feedback for detected gestures

> ⚠️ Prototype: Work in progress

## Development

```sh
npm install
npm run dev
```
