# Hand Gesture Dashboard

A smart dashboard controlled by hand gestures via webcam. Built with React, TypeScript, and MediaPipe.

**Features:**

* [ ] **Widgets:**

  * [x] Date/Time
  * [x] Weather
  * [x] iCloud Calendar Integration
  * [ ] Reminders *(planned)*
  * [ ] Spotify integration with playback controls *(in progress)*
* [x] **Gesture Recognition:**

  * [x] Static gesture detection using webcam and gesture recognition model
  * [x] Processing pipeline for detecting temporal gestures using landmarks and static gesture states

    * [x] Point-and-click gestures for cursor control and interacting with elements
    * [x] Swipe gestures for moving between screens
* [x] **Visual Feedback:**

  * [x] Visual cues for detected gestures

> ⚠️ Prototype: Work in progress
